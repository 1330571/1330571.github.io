C++提供的文件相关的标准库的简易使用方法

C++ Filesystem library

该文件系统库可能会因为操作系统的实现方式的不同产生不同的结果，如FAT系统没有软链接(Symbolic Links)和多份硬链接(Hardlinks)，如果同时有多个线程、进程、计算机操作同一个文件系统对象，则行为为未定义(undefined)

文件一共分为以下几种类型：

  • 文件夹(directory)
  • 常规文件(regular file)
  • 硬链接(hard link)
  • 软链接(symbolic link)

文件路径也分为三种：

  • 绝对路径(absolute path)
  • 相对路径(relative path)
  • canonical path

目标

该文章旨在利用C++标准库filesystem实现一个计算文件夹大小的功能，获取一下所有大于512MB的文件夹，先看一看程序的运行效果

常用

###path类

用来表明一个文件路径。

可以是:

  • 绝对路径，如/usr/local/bin/
  • 相对路径，如./

可以通过地址去构造一个path类

directory_iterator类

非递归的遍历所有文件，即跳过文件夹

recursive_directory_iterator类

相比directory_iterator可以同时遍历文件夹

directory_entry类

从iterator类中用来遍历的迭代器，包含文件路径，额外的文件信息(包括文件类型，文件权限，文件大小…)

获取文件类型的函数通常是is_xxx开头
