<=>

基本使用

注释中的方法也是可以用的，类型为std::__1::strong_ordering res

  • (a <=> b) < 0 if lhs < rhs
  • (a <=> b) > 0 if lhs > rhs
  • (a <=> b) == 0 if lhs and rhs are equal/equivalent.

如果操作数中有浮点数，这是std::partial_ordering，多一个unordered结果，如选择NaN和其他数比较

See stack overflow

Section 2.2.3 of Herb Sutter’s Consistent comparison paper talks about the use of <=> with other data types of the language where subtraction may produce inconsistent results.

Heap & Stack

  • Heap

动态内存分配，容量大，根据需要分配/释放，速度慢

  • Stack

静态内存分配，容量小，自动分配/释放，速度快

如果定义一个vector<>类型

对象会被定义在Stack上，而存储的数据会被分配在Heap上
